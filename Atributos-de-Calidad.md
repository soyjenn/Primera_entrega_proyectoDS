|Requerimiento|Categoría|Descripción|
|-|-|-|
|ADC1|Rendimiento|El sistema debe responder a cualquier consulta en menos de 2 segundos.|
|ADC2|Rendimiento|El sistema debe poder manejar al menos 500 registros sin degradación perceptible.|
|ADC3|Usabilidad|La interfaz debe ser intuitiva y fácil de usar para personal administrativo sin experiencia técnica.|
|ADC4|Usabilidad|Los mensajes de error deben ser claros y comprensibles.|
|ADC5|Mantenibilidad|El código debe estar estructurado de forma modular para permitir la evolución futura.|
|ADC6|Mantenibilidad|Se debe incluir documentación técnica básica (diagramas y descripción de clases).|
|ADC7|Portabilidad|El sistema debe poder ejecutarse en cualquier equipo con Java instalado (sin dependencias externas pesadas).|
|ADC8|Seguridad|Solo usuarios autorizados pueden registrar o eliminar ventas y productos.|
|ADC9|Seguridad|Los datos deben almacenarse de forma segura y con respaldo local.|
